# 📖 Overview

ActionPilot AI is a modern AI-powered productivity platform designed to transform the way people manage work.

Instead of functioning as another reminder application, ActionPilot AI acts as an intelligent execution partner that continuously analyzes tasks, voice notes, emails, uploaded documents, and user activity to determine what should be done next.

Using Google's Gemini AI, the platform proactively assists users by extracting actionable tasks, prioritizing workloads, generating execution plans, summarizing progress, and helping ensure deadlines are met before they become critical.

Whether you're a student managing assignments, a professional balancing projects, a freelancer handling multiple clients, or an entrepreneur building a business, ActionPilot AI is designed to reduce decision fatigue and increase productivity.

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/FaceHexOfficial/ActionPilot-AI.git
```

Navigate

```bash
cd ActionPilot-AI
```

Install packages

```bash
npm install
```

Create

```text
.env.local
```

Add

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run

```bash
npm run dev
```
